# Database-Management-System

## 📌 Overview

This project is a relational database system designed to manage course information, students, instructors, and enrollments.

The database was developed using SQL and demonstrates the design and implementation of a normalized relational database with efficient data retrieval and management.

---

## 🎯 Objectives

- Design a relational database.
- Create database tables with appropriate relationships.
- Apply Primary and Foreign Keys.
- Maintain data integrity.
- Perform CRUD operations.
- Execute SQL queries for data analysis.
